NEW HTML ATTRIBUTES (EASY TO USE)
─────────────────────────────────────────────────────────────────────────────

data-floating-card
  → Add to any card element for interactive 3D hover
  Example: <div data-floating-card>Card content</div>

data-depth-layer="1|2|3"
  → Creates parallax layers (1=back, 3=front)
  Example: <div data-depth-layer="1">Background</div>

data-text-3d
  → Adds animation to text elements on scroll
  Example: <h2 data-text-3d>Your Heading</h2>

class="float-3d"
  → Continuous floating animation (6 second loop)
  Example: <div class="float-3d">Content</div>

class="card-tilt"
  → Simple 3D tilt effect on hover
  Example: <div class="card-tilt">Content</div>

class="glow-3d"
  → Glowing drop-shadow effect
  Example: <div class="glow-3d">Content</div>

TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────

Cards not responding?
  → Check data-floating-card attribute is present
  → Open DevTools (F12) for JavaScript errors

Parallax not working?
  → Use data-depth-layer="1|2|3"
  → Try scrolling the page
  → Check element has non-zero dimensions

Text not animating?
  → Add data-text-3d to heading element
  → Element must scroll into viewport
  → Wait for animation to trigger on scroll

Performance issues?
  → Reduce number of floating cards
  → Check browser FPS counter
  → Disable on low-end mobile devices
